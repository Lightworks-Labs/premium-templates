# Traceability Matrix

A requirements traceability matrix for linking system requirements to design outputs, verification activities, and test cases — supporting IEC 62304, ISO 13485, and 21 CFR Part 820 compliance.

## What's included

- **Traceability Matrix database** — track requirements with verification method, status, risk level, and linked test cases

## Usage

Install this bundle to add a `/traceability-matrix` database to your repo. Each record represents a requirement, linked to its verification evidence and approval status.
